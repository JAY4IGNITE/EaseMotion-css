# Theme Switcher with CSS Variables

## Description

This demo illustrates how CSS custom properties can be used to create a simple light/dark theme switcher.

## Files

- demo.html
- style.css

## Features

- CSS Variables
- Light Theme
- Dark Theme
- Smooth transitions
- Minimal JavaScript

## Usage

1. Open `demo.html`.
2. Click **Toggle Theme**.
3. Observe the color changes driven by CSS variables.

## Concepts Used

- CSS Custom Properties
- Theme switching
- DOM class toggling
- CSS transitions
